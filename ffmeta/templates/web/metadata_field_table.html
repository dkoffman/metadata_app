Metadata FieldDescription
NameThe name of the variable in the data
Old NameThe name of the variable in the data before the re-release
LabelA short description of the variable
Question TextThe exact wording of the question from the questionnaire
ProbeThe exact wording of the lead-in question from the questionnaire, where applicable
Data SourceA categorical variable, defining where the data stored in the variable was generated from
SurveyThe subject of the survey that the variable was from
WaveThe round of data collection that this variable was in
RespondentThe person that provided the information for this variable
Focal PersonThe subjects of the variable
Topic (1,2)The broader sociological concept(s) that are measured by this variable
Sub-Topic (1,2)The specific sociological concept(s) that are measured by this variable
ScopeThe size of the sample that this question was asked in
Data TypeThe way that the data is stored in the variable, e.g binary, categorical, continuous
WarningA code to alert the user that this variable may have an unexpected way to store response code
