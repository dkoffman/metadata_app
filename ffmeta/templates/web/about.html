Available Fragile Families Metadata

Descriptions of the available metadata:

new_name: Corresponds to what the variable was named after the re-release of February 2018 .

varlab: A description of the variable, generated from the variable label in the Stata version of the data.

type: Describes the way the data is stored. Can be one of five categories: Binary (bin), Continuous (cont), Unordered Categorical (uc), Ordered Catergorical (oc) or String (string).

group: Links the variable to similar variables in other instruments.

q_group_N: The number of similar variables in the given variables group.

topic[1,2]: A tag that describes the specific concept measured by the variable.

umbrella[1,2]: A more general tag describing the concept measured by the variable.

scope: An indicator of how many cities were asked the question corresponding to this variable.

warning: An indicator that this variable may have data stored in a way that would be unusual given its type.

source: The original source of the data, i.e questionnaire, constructed variable, etc.

respondent: The focus of the source, i.e. father/mother/PCG/. Generated from the varname.

wave: Which wave of data collection the variable comes from. Generated from the varname.

section: Which section of the instrument the variable comes from. Generated from the varname.

leaf: The question number or short descriptor, depending on the source. Generated from the varname.

q_group_list: A string list seperated by spaces, containing all of the names of similar variables.
