Welcome to the FFCWS Metadata Explorer!

What you will find on this website:

  1. Browse Variables -- This resource is great for getting started with exploring the FFCWS variables. Select "Browse Variables" in the menu above to sort FFCWS variables by topic, wave, respondent, focal person, scales, survey, source, and/or variable type
  2. Advanced Search -- Select "Advanced Search" in the menu above to perform more detailed queries and/or text searches in the FFCWS metadata
  3. Download full metadata -- Click the link in the menu above to download the full metadata as a CSV file.

Other API resources to help you use the metadata:

  1. FFCWS Metadata API -- Click here for instructions on how to directly access the Web API without using the website interface
  2. R Package-- This package allows R users to retrieve metadata within R via the FFCWS Metadata API. Users can select, filter, and search metadata for relevant variables. More information is available on github
  3. Python Package -- This Python package provides convenient wrrappers to interface with the FFCWS Metadata API in Python. By downloading and importing the single ff.py module, users can query the metadata variables.
